overflow-wrap property
specifies whether or not the browser can break lines with long words, if they overflow the container. 
                            before (overflow-wrap: normal)

                            Lorem ipsum dolor sit amet consectetur.
                        

                            after (overflow-wrap: break-word)
                            
                            Lorem ipsum
                            dolor sit
                            amet
                            consectetur
appearance property
used to control the appearance of form controls

before (appearance: auto)

after (appearance: none)

attribute selector
used to select elements with a specified attribute
.input["text"] { background-color: red; }
checked input attribute
to force an input type checkbox to be checked after the html loaded

before (<input type="checkbox">)

before (<input type="checkbox" checked>)

:checked input puesdo class
selector matches every checked <input> element
input["checkbox"]:checked { width: 20px; height: 20px; }
